#My Photographic Site
This is a simple website showcasing my favourite photos.
It include a photo gallery ,a short biography section,and a design
overview.The project is part of my learning journey in web development.

## Features
The website showcases basic web design concepts such as:
page structure using HTML and Styling CSS 

Responsive photo gallery with hover effects
Short biography about the website
Design attached as as image for review
Custom CSS styling with background colors,fonts and layout
